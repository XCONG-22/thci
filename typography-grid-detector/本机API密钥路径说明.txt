================================================================================
推荐：用「普通文件名」填密钥（记事本可正常打开、纯文本）
================================================================================

路径（与 package.json 同一文件夹）：

  d:\生成设计cursor\cursor-test-extracted\cursor-test-main\typography-grid-detector\api-credentials.env

若还没有这个文件：
  - 双击运行一次「启动开发.bat」会自动从 api-credentials.env.example 复制出来；
  - 或手动把 api-credentials.env.example 复制一份改名为 api-credentials.env。

用 Windows「记事本」打开 api-credentials.env，在等号后面写密钥，例如：

  OPENAI_API_KEY=sk-你的密钥

保存即可（编码选 UTF-8）。不要把这个文件发给任何人或上传到公开 Git 仓库。

--------------------------------------------------------------------------------
可选：.env.local
--------------------------------------------------------------------------------

若你的编辑器能正常把 .env.local 当文本编辑，仍可使用；启动时会最后加载 .env.local，
同名变量会覆盖 api-credentials.env。

--------------------------------------------------------------------------------
访问地址
--------------------------------------------------------------------------------

  http://localhost:3010

（npm run dev / 启动开发.bat 已固定开发端口 3010，避免与本机其它占用 3000 的程序混淆。）
