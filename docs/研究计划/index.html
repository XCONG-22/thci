研究陈述（300–400字）

以导师阅读为导向的凝练版

本研究聚焦年轻人与中年人睡眠质量下降及伴随的情绪困扰，面向“非药物、可持续、自我调节”的健康需求，探索生成式AI与艺术疗愈在健康视觉传达设计中的可行路径。研究以睡眠数据（节律、时长、觉醒等）与情绪自评为输入，建立参数可追溯的内容生成机制，生成动态自然景观、抽象疗愈图案与柔和渐变等低刺激视觉内容，并将其嵌入睡前放松引导、夜间温和唤醒与晨间情绪重置的交互流程。设计层面重点优化夜间可读性、信息层级、色彩情绪一致性与审美统一，形成分龄差异化的界面策略（年轻人强调沉浸与个性表达，中年人强调可读与确定反馈）。最终通过原型系统与用户测试，从可用性、主观放松度/唤醒度与睡眠相关自评等维度验证个性化AI艺术疗愈的干预价值，为健康视觉传达提供方法框架与证据支持。

备注：正文刻意避免“概念化修辞”，强调可解释生成、夜间使用约束与可验证性，以符合导师审阅习惯。

问题与背景

为何值得做｜设计与证据缺口

现实问题
  • 睡眠问题常伴随焦虑、压力与情绪波动，需求具有高频与长期性。
  • 数字化助眠产品“同质化内容”明显，个体差异与情境变化难以覆盖。
  • 夜间使用对亮度、闪烁、动效节奏更敏感，稍有不当就会反向刺激。
研究缺口（设计视角）
  • 艺术疗愈的“情感价值”常被描述，却缺少面向界面的可操作设计变量。
  • 生成式AI用于助眠多停留在“内容生成”，对交互流程与信息架构关注不足。
  • 缺乏把“个性化生成”转化为可验证体验收益的评估框架。

研究目标与研究问题

明确边界｜可回答

目标
  • 构建“数据/情绪 → 生成内容 → 交互流程 → 体验与指标”的设计框架。
  • 提出分龄视觉与交互策略，形成可复用的界面规范与内容库结构。
  • 以原型与用户研究验证个性化生成对放松与睡眠体验的增益。
研究问题
  • 睡眠数据与情绪状态应如何映射为低刺激、可解释的视觉参数？
  • 夜间界面如何在“疗愈美学—可读性—交互负荷”之间取得稳定平衡？
  • 个性化生成在何种条件下显著优于通用内容？其证据如何建立？

方法与技术路线（概要）

从内容到界面再到验证

A. 内容生成（可解释约束）
  • 输入：节律/觉醒/时长等睡眠特征 + 情绪自评 + 个体偏好（色系/动感/明暗）。
  • 输出：动态自然景观、抽象图案、柔和渐变等多模态视觉内容。
  • 约束：低亮度、低闪烁、低频动效；参数—视觉变化可追溯，便于迭代与研究。
B. 交互与视觉传达（分龄策略）
  • 流程：睡前放松引导 → 夜间温和唤醒 → 晨间情绪重置。
  • 视觉：信息层级与对比控制、字体可读、色彩情绪一致、动效节奏克制。
  • 分龄：年轻人偏沉浸与自我表达；中年人偏“清晰、可控、低负荷”。
C. 评估与验证（面向证据）
  • 可用性：任务完成、错误率、主观负荷（如 NASA-TLX 或简化量表）。
  • 情绪体验：放松度/唤醒度、情绪稳定性自评，必要时加访谈获取机制解释。
  • 睡眠体验：入睡容易度、夜间觉醒感受、醒后恢复感等自评；可加入可穿戴数据作为辅助证据。
  • 对照：通用内容 vs 个性化生成；不同视觉风格/动效节奏的设计对照。

预期创新点

设计贡献与研究贡献

设计层面
  • 将艺术疗愈转译为可操作的视觉变量与界面规范（亮度、对比、频率、节奏、层级）。
  • 提出分龄交互策略与夜间使用约束的系统化方法。
研究层面
  • 构建“个性化生成—体验收益”的评估框架与对照实验路径。
  • 强调可解释生成（可追溯参数）以支持迭代与学术论证。

预期产出

导师最关心的可交付

设计产出
  • 可交互原型（核心流程 + 个性化生成展示）。
  • 视觉语言与组件规范（夜间可读/动效节奏/色彩情绪）。
  • 疗愈内容库与生成参数模板（可复用、可迭代）。
研究产出
  • 研究框架与方法论总结（适用于健康视觉传达语境）。
  • 用户研究与验证报告（指标、对照、结论与局限）。
  • 可推广设计建议（面向同类非药物睡眠干预产品）。

进度与验证计划（建议）

可按学期周次调整

  • 阶段1（调研）：睡眠健康与艺术疗愈相关文献梳理；竞品与夜间界面规范分析；访谈提炼需求。
  • 阶段2（框架）：确定数据/情绪特征与视觉参数映射；定义内容生成约束与风格边界；信息架构与交互流程草案。
  • 阶段3（原型）：完成关键界面与交互原型；形成视觉组件与动效节奏规范；制作多方案对照版本。
  • 阶段4（测试）：可用性测试 + 体验量表 + 半结构访谈；必要时增加短周期日记研究。
  • 阶段5（总结）：迭代收敛设计；完成论文/说明书与成果展示页面。
